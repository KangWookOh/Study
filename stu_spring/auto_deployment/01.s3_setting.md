# S3 세팅

S3란 Simple Storage Service의 약자로 **파일 서버의 역할을 하는 서비스**이다.

<br>

배포 자동화를 위한 Code Deploy는 S3에서 Zip파일이 올라오면 그것을 감지해서 배포를 시작하기 때문에 S3를 사용해야 한다.

우선 AWS에 접속해서 S3를 검색해 준다.
