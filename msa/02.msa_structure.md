## API Gateway란

API Gateway는 요청의 최 상단에서 인스턴스로 연결해 주는 Proxy Server와 비슷한 역할을 한다.

> Proxy 서버는 라우팅 기능과 약간의 보안 기능이 전부이지만, API Gateway는 좀 더 큰 범위이다.

API Gateway를 사용함으로써 **모든 API들의 엔드포인트가 단일화**될 수 있다.

API에 대한 인증과 인가 기능 또한 포함하고 있다.
